# Framer-Motion

- install `npm i motion`
- Basic introduction to Framer Motion with a 3D hover effect on a button using React and Tailwind CSS `content.tsx`.

## some useful properties of motion component
- `whileHover`: Defines the animation to apply when the element is hovered over.
- `whileTap`: Defines the animation to apply when the element is being clicked or tapped.
- `transition`: Specifies how the animation should progress over time, including duration and easing.
- `style`: Used to set initial styles or transform properties that are not animated.
- `initial`: Sets the initial state of the animation before any interaction.
- `animate`: Defines the animation to apply when the component mounts or when its state changes.
- `boxShadow`: Used to create shadow effects around elements, enhancing the 3D appearance during interactions.
- `translateZ`: A CSS transform property that moves an element along the Z-axis, creating a sense of depth in 3D space.
- `rotateX` and `rotateY`: CSS transform properties that rotate an element around the X-axis and Y-axis, respectively, contributing to the 3D effect.
